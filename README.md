# World Cities

Cities of the World in Monochrome!


## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You will need IDE like VSCode or Atom Editor to edit files in this project.

### Installing

1) Clone the Project

```
Go To Project Page and Click Green Clone or Download Button on Github.
```

2) Setting Up the Github Desktop

```
After you downloaded Github Desktop or Git Bash and you logged in with your profile, clone the repository using the link you got on directory you want.
```

3) Opening Files

```
Open photosite.html with VSCode.
```

4) Running Files

```
Running photosite.html is enough
```
